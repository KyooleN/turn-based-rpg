# KrystalGem

Salve família! Esse jogo é fruto de um trabalho da faculdade da matéria de Algorítmos e Estruturas de dados. Se trata de fazer uma mecânica de um RPG de turno. Esperamos que gostem! 

# Como rodar o jogo?

 Clone o repositório
 
 Digite ` g++ *cpp -o game `
 
 Digite ` ./game  `
 
 Divirta-se.
